# EXAM_prep_ISC2_CC
A modern, interactive practice exam platform designed to help learners prepare for the ISC2 Certified in Cybersecurity (CC) certification.
This project provides a realistic exam‑style experience with timed tests, question navigation, scoring, history tracking, and detailed review features.

# FEATURES
1. 130 curated ISC2 CC practice questions
2. Timed or untimed exam modes
3. Quick, Standard, and Full test presets
4. Dynamic question navigation grid
5. Flag questions for review
6. Instant scoring and performance breakdown
7. Detailed review of all questions after submission
8. Local history tracking (no backend required)
9. Dark futuristic cybersecurity UI (neon theme)
10. Responsive design for mobile and desktop

# Tech Stack
1. HTML5 – Structure
2. CSS3 – Custom dark cyber UI + animations
